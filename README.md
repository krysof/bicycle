# 今天也要送到

面向老年玩家的 3D 网页陪伴型送报游戏原型。

## 在线访问

GitHub Pages：

https://krysof.github.io/bicycle/

## 本地运行

```bash
npm run start
```

然后打开：

```text
http://localhost:5173
```

## 项目结构

```text
docs/              企划案与开发计划
src/data/          邻居、问卷等数据
src/game/          难度、移动、投递规则
src/input/         输入处理
src/render/        Canvas 等距 3D 绘制
src/state/         状态与本地保存
src/ui/            页面与 HUD
src/app.js         游戏流程控制
src/main.js        启动入口
```
